# Data

This directory contains all data files for the Bravura project. Data is not tracked in git.

## Structure

### `cohort_a/`
- `aggroPerformance.xlsx` — Per-trial shock decisions, latencies, choices (126 subjects x 30 trials)
- `latPerformance.xlsx` — Clean latency data (124 subjects)
- `beliefs.xlsx` — Belief ratings per opponent (0-10 scale)
- `vba_input.xlsx` — Transposed input for MATLAB VBA model fitting
- `coefficients.csv` — BMA-averaged model coefficients [Kr1, Krc, Kp, Kwc]
- `predictions.csv` — Predicted P(shock) per trial
- `decisions.csv` — Actual binary shock decisions
- `fit_metrics.csv` — R2, accuracy, balanced accuracy, log evidence
- `subject_ids.csv` — Subject ID list
- `free_energy_matrix.csv` — Per-model free energies
- `vba_posteriors.mat` — muPhi and SigmaPhi per subject (for MC uncertainty)
- `outliers.mat` — 12 excluded subject IDs
- `physPerformance.xlsx` — Physiological features (HR, HRV, respiration, EDA) across experimental blocks
- `cov_stats.mat` — Simulation-recovery: condition numbers and determinants
- `corr_preds.mat` — Simulation-recovery: per-subject parameter correlation matrices

### `cohort_b/`
Same structure as `cohort_a/`. Beliefs on 0-5 scale (rescaled to 0-10 in notebooks).

### `shared/`
- `AggressionScript.xlsx` — Preprogrammed opponent behavior (shocks, wins per trial)
- `trial_events.csv` — Parsed per-trial data from BioPac events (duration, choice, outcome)

### `processed/`
Pipeline outputs (generated by notebooks):
- `behav_Xa.csv`, `behav_Xb.csv` — Behavioral features with cluster labels
- `delta_hr_long.csv`, `delta_hr_long_b.csv` — Delta-HR long format for brms
- `baseline_hr.csv` — Baseline HR per subject
- `shock_latency_long.csv` — Shock latency per trial (shock events only)
- `shock_long.csv` — Shock counts long format for brms
- `psap_ilr.csv` — PSAP compositions (noise-replaced)
- `physio_cardiac_long.csv` — HR + varimax HRV components for brms
- `physio_multivariate_long.csv` — 6-DV physio data for brms

### `brms/`
Cached brms model fits and outputs:
- `shocks/` — Binomial shock model
- `psap/` — Dirichlet PSAP model
- `delta_hr/` — Delta-HR mixed model (Cohort A)
- `delta_hr_replication/` — Replication BFs (BayesRep)
- `baseline_hr/` — Baseline HR model
- `physio_cardiac/` — Multivariate cardiac model (HR + HRV RCs)
- `physio_multivariate/` — 6-DV multivariate physio model
- `latency/` — Shock latency model

### `old/`
Intermediate files from old pipeline (reference only).

### `raw/`
Raw data from AggressionProjectDataShare (MatlabEvents, physio toolboxes).
